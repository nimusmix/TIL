# Web_03_bootstrap

## ✨ Spacing (margin and padding)

- `{property}` `{sides}`-`{size}`
- `property`
    - **m** : margin
    - **p** : padding
- `sides`
    - **t** : top
    - **b** : bottom
    - **s** : start
    - **e** : end
    - **x** : left & right
    - **y** : top & bottom
    - **blank** : all 4 sides
- `size`
    - **0**
    - **1** : 0.25rem / 4px
    - **2** : 0.5rem / 8px
    - **3** : 1rem / 16px
    - **4** : 1.5rem / 24px
    - **5** : 3rem / 48px
    - **auto** : center
